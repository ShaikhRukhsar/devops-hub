Linuc Commands
File & Directory
ls            # list files
ls -l         # long details
ls -a         # show hidden
pwd           # show current path
cd /path      # change directory
cd ..         # go back

mkdir test            # create directory
mkdir -p a/b/c        # create with parent dirs
rmdir dir             # remove empty dir
rm file               # remove file
rm -r dir             # remove directory recursively
rm -rf dir            # force delete

cp file1 file2        # copy file
cp -r dir1 dir2       # copy directory
mv a b                # move/rename
touch file            # create empty file

View & Edit Files
cat file              # view file
cat file | less       # view page by page
head file             # first 10 lines
tail file             # last 10 lines
tail -f file          # live log monitoring


View last 100 lines:

tail -n 100 file


Edit files:

nano file
vim file

Search

Search text inside file:

grep "word" file
grep -i "word" file   # ignore case
grep -r "word" /path  # recursive search


Find files:

find /path -name filename
find /var/www -type f -name "*.log"

Permissions
chmod 755 file
chmod u+rwx file
chown user:group file


Permissions breakdown:

7 = rwx

5 = r-x

4 = r--

System Info
uname -a              # system info
hostname              # show hostname
date                  # show date
uptime                # system load & running time
whoami                # current user

Disk & Storage
df -h                 # disk usage
du -sh *              # size of folders in current dir
mount / umount        # mount/unmount storage

Process & Performance
top                   # live processes
htop                  # better top (if installed)
ps aux                # list all processes
kill <pid>            # stop process
kill -9 <pid>         # force stop


Search process:

ps aux | grep nginx
pgrep nginx


Kill by name:

pkill nginx

Network
ip a                  # show IP
ifconfig              # show IP (older)
ip r                  # routing table
ping google.com       # check connectivity
netstat -tulnp        # ports & services
ss -tulnp             # newer netstat
curl http://localhost # check web server

Package Management

Ubuntu/Debian:

sudo apt update
sudo apt install nginx
sudo apt remove nginx


RHEL/CentOS:

sudo yum install nginx
sudo yum remove nginx
sudo systemctl status nginx

Services
systemctl start nginx
systemctl stop nginx
systemctl restart nginx
systemctl enable nginx
systemctl status nginx

Archiving & Compression
tar -cvf backup.tar dir/      # create archive
tar -xvf backup.tar           # extract
tar -czvf backup.tar.gz dir/  # create compressed archive
tar -xzvf backup.tar.gz       # extract

Q18: How to check ports and services?
A: ss -tulnp.

Q19: How to copy a file from local to remote?
A: scp file user@host:/path.

Q20: How to sync directories efficiently?
A: rsync -av source/ destination/
